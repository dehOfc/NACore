# N.ACORE - Precisão em cada solução

Repositório oficial da **N.ACORE**, uma consultoria tecnológica especializada em automação inteligente, presença digital e escala de vendas.

## 🚀 Sobre o Projeto
Este é um ecossistema digital completo desenvolvido para demonstrar autoridade técnica e converter leads através de ferramentas interativas e conteúdo de valor.

### Principais Funcionalidades
- **Landing Page de Alta Performance**: Focada em conversão com design premium e glassmorphism.
- **Blog Técnico**: 4 artigos aprofundados sobre Automação, IA, n8n e Marketing Digital.
- **Calculadora de ROI**: Ferramenta interativa que estima a economia mensal e anual com automação.
- **Chatbot Integrado**: Widget conversacional para qualificação automática de leads.
- **Formulários Inteligentes**: Validação em tempo real e máscaras de entrada.

## 🛠️ Tecnologias Utilizadas
- **Frontend**: HTML5 Semântico, Vanilla CSS3 (Custom Properties), JavaScript ES6+.
- **Ícones**: Boxicons.
- **Tipografia**: Outfit & Inter (Google Fonts).

## 📂 Estrutura do Repositório
```
/
├── assets/             # Logos e imagens
├── blog/               # Páginas de artigos e listagem do blog
├── index.html          # Landing page principal
├── style.css           # Estilos globais
├── blog.css            # Estilos específicos do blog
├── widgets.css         # Estilos da calculadora e chatbot
└── script.js           # Lógica de interatividade e ferramentas
```
